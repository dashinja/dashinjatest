CSS Background Shorthand

	Background has the following properties:
		
		Background-Color
		Background-Image
		Background-Repeat
		Background-Attachment
		Background-Position

		Background Short Hand Follows this Order: CIRAP monster (Sounds like "Syrup") and is denoted as the following:

			selector {
				background: C I R A P; 
			}

		where CIRAP monster is treated as a variable, and substituted into each of the places is the VALUE
		of the given property, as opposed to a "property: value" pair as per usual.

		An example of this with actual property values is as follows:
			body {
				background: red no-repeat fixed top right;
			}

		Notice, the property names are foregone on account of CSS parsers expecting a particular order of 
		property VALUES given.

CSS Text Properties

The css text property has at least 14 different variations. Please Note: that when using the color property it is best practice to also include a "background-color" specification as well, especially for W3C compliant css.


Please note: The tag must be used immediately after the "table" tag.

CSS Text Properties
Property Description
color Sets the color of text
direction Specifies the text direction/writing direction
letter-spacing Increases or decreases
line-height Sets the line height
text-align Specifies the horizontal alignment of text
text-decoration Specifies the decoration added to text
text-indent Specifies the indentation of the first line in a text-block
text-shadow Specifies the shadow effect added to text
text-transform Controls the capitalization of text
text-overflow Specifies how overflowed content that is not displayed should be signaled to the user
unicode-bidi Used together with the direction property to set or return whether the text should be overriden to support multiple languages in the same document
vertical-align Sets the vertical alignment of an element
white-space Specifies how white-space inside an element is handled
word-spacing Increases or decreases the space between words in a text
